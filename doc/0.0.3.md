# v0.0.3 — "多语言输入 + Azure TTS"

## 功能

支持任意语言输入，自动翻译成中文；弃用 Web Speech API，改用 Azure 云端 TTS，实现跨平台一致的语音合成。

| 功能 | 说明 |
|------|------|
| 自动语言检测 | Azure Translator 自动识别输入语言（日语、韩语、法语、德语等） |
| 任意语言→中文 | 翻译结果与 v0.0.1/0.0.2 一致（简体+繁体+拼音） |
| 中文语音 | 固定使用 `zh-CN-XiaoxiaoNeural`（女声神经语音，跨平台一致） |
| 多语言语音 | 根据检测到的源语言自动选择对应神经语音（日语 Nanami、韩语 SunHi 等） |

---

## 技术实现

### Azure Translator（替换 DeepL）

REST API：`https://api.cognitive.microsofttranslator.com/translate?api-version=3.0&to=zh-Hans`

- 不传 `from` 参数，自动检测源语言
- 返回 `detectedLanguage.language`（小写，如 `"en"`, `"ja"`）
- 免费额度：F0 套餐 2M 字/月，无需绑信用卡

### Azure Speech TTS（替换 Web Speech API）

REST API：`https://{region}.tts.speech.microsoft.com/cognitiveservices/v1`

- 输入：SSML，指定神经语音名称
- 输出：MP3 音频（`audio-24khz-48kbitrate-mono-mp3`）
- 后端 `GET /api/tts?text=...&lang=zh-CN` 代理，返回 MP3 Buffer
- 免费额度：F0 套餐 500k 字/月

### 前端播放

`SpeakerButton` 改用 `fetch('/api/tts')` + `new Audio(blobUrl)`：
- 速度控制：`audio.playbackRate = rate`
- 移除：`splitByLang()`, `pickZhVoice()`, `pickEnVoice()`, `SpeechSynthesisUtterance`

## 已知限制

- Azure Speech F0 免费套餐每月 500k 字，超出后需付费（~$16/百万字）
- `/api/tts` 每次点击发起网络请求，有轻微延迟（首次约 500ms）
- 速度控制通过 `audio.playbackRate` 实现，不影响音质
