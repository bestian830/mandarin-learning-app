# v0.0.5 — "中文分词 + 词性悬浮卡"

## 功能

将译文拆分为词语单位，鼠标悬浮时显示词性、拼音、繁体信息卡。

| 功能 | 说明 |
|------|------|
| 中文分词 | 使用 nodejieba tag() 对译文按词切分 + ICTCLAS 词性标注 |
| 词性分类 | 将 32+ 种 ICTCLAS 标签映射为 11 个学习者友好分类 |
| 悬浮卡片 | 鼠标悬停词语时，在右下方弹出卡片，显示词语、拼音、词性（中/英）、繁体 |
| 分词修正 | 用户词典覆盖基础词典误判（如"颜色"被混入专有名词"种颜色"） |

## 词性分类

| 分类 key | 英文 | 中文 | 颜色 | ICTCLAS 标签 |
|----------|------|------|------|-------------|
| verb | Verb | 动词 | 红 | v, vd, vn, vg |
| noun | Noun | 名词 | 绿 | n, nz, ns |
| proper | Proper | 专有名词 | 紫 | nr, nt |
| pronoun | Pronoun | 代词 | 蓝 | r, rg |
| adverb | Adverb | 副词 | 橙 | d, dg |
| adj | Adj. | 形容词 | 黄 | a, ad, an, ag |
| particle | Particle | 助词 | 灰 | u, ug, uj, ud, ul, uz, uv, y, e |
| time | Time | 时间词 | 青 | t, tg, f |
| number | Num. | 数量词 | 黄绿 | m, mg, mq, q |
| conj | Conj. | 连词 | 深灰 | c, p |
| idiom | Idiom | 成语 | 深紫 | i, l |

标点符号（w, x）不显示悬浮卡。

## 问题根因与修复

### "哪种颜色" 误分为 "哪" + "种颜色"

- **根因**：nodejieba 基础词典将 "种颜色" 收录为专有名词（nr），统计权重高于 "哪种" + "颜色" 的切分方案
- **现象**：悬浮卡显示"种颜色"为专有名词；"哪"单独成词
- **失败尝试**：`insertWord("哪种", freq, tag)` — 第二参数为数字时抛出 "A string was expected"；单参数 `insertWord("哪种")` 返回 true 但不足以覆盖基础词典权重
- **修复**：创建 `src/lib/user_dict.utf8`，以 `100000` 高频显式声明"颜色"为名词、常用指示代词复合形式为代词，调用 `nodejieba.load({ userDict })` 覆盖基础词典

### 悬浮卡遮挡后方文字

- **根因**：卡片定位 `left-full top-1/2`，向右展开，覆盖紧接在后的汉字
- **修复**：改为 `top-full left-0 mt-2`，卡片出现在词语正下方，不干扰行内其他字

## 技术实现

### 后端：`segmentText(text, chars)`

```typescript
// 用户词典覆盖基础词典误判
load({ userDict: join(__dir, "user_dict.utf8") });

// nodejieba tag() 返回 [{word, tag}]
// tag 映射到 11 个学习者分类，标点归 "punct"（不显示卡片）
const category = PUNCT_TAGS.has(t) ? "punct" : (TAG_MAP[t] ?? "other");
```

每个 Segment 包含：`word`, `traditional?`, `pinyin`, `tag`, `category`, `chars[]`

### 前端：`<SegmentGroup seg={seg} />`

```tsx
// 纯 CSS hover，支持鼠标滑入卡片后继续悬停
<div className="relative group/seg">
  <div className="flex">{charNodes}</div>
  <div className="absolute top-full left-0 mt-2
                   invisible opacity-0
                   group-hover/seg:visible group-hover/seg:opacity-100
                   transition-[opacity,visibility] duration-150 z-30">
    {/* 卡片内容：词语 + 繁体 + 拼音 + 词性徽章 */}
  </div>
</div>
```

使用 `invisible/visible`（而非 `pointer-events-none`）使鼠标可以从词滑入卡片保持显示。
