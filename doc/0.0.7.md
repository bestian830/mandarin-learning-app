# v0.0.7 — i18n 架构升级（Azure 自动翻译 + 25 语言）

## 功能

| 功能 | 说明 |
|------|------|
| 语言切换器 | 右上角下拉菜单，25 种语言按使用人数排序，中文置末 |
| 自动翻译 UI | 只维护英文源字符串，其他 24 种语言由 Azure Translator 批量翻译 |
| 双层缓存 | SQLite（后端永久）+ localStorage（前端永久），首次约 300-500ms，之后瞬开 |

| 浏览器语言检测 | 首次访问自动匹配浏览器语言偏好 |

## 架构设计

### 核心原则：只维护英文

```typescript
// backend/src/routes/i18n.ts — 唯一需要维护的地方
const SOURCE_EN = {
  subtitle:    "Think it in your language. Say it in Chinese.",
  placeholder: "What's in your thought?",
  translating: "Translating",
  errorTitle:  "Translation failed",
  chip1: "apple", chip2: "thank you",
  chip3: "I love you", chip4: "Good morning", chip5: "friend",
};
```

新增 UI 字段：只在 `SOURCE_EN` 加一行，所有语言自动翻译。
新增语言：前端 `LANGUAGES` 数组加一行，零翻译工作。

### 数据流

```
用户切换语言
    ↓
前端检查 localStorage（mm_i18n_{lang}）
    ↓ 未命中
GET /api/i18n?lang=vi
    ↓
后端检查 SQLite i18n_cache 表
    ↓ 未命中
Azure Translator 批量翻译（~9条字符串，一次请求，约 300-500ms）
    ↓
写入 SQLite 缓存 → 返回
    ↓
前端写入 localStorage → 渲染
```

### 缓存策略

| 层级 | 位置 | 有效期 | 读取速度 |
|------|------|--------|---------|
| L1 | localStorage | 永久 | 0ms（同步） |
| L2 | SQLite i18n_cache | 永久 | <5ms |
| L3 | Azure Translator | — | 300-500ms（首次） |

如需强制刷新翻译（如源字符串修改后）：清除对应 localStorage key 或删除 SQLite 中该 lang 行。
