# v0.0.9 — 拼音教学侧边栏（Pinyin Guide）

## 功能概述

实装汉堡菜单打开的「Pinyin Guide」侧边栏，提供系统化的拼音教学内容，支持点击播放标准普通话发音，完全随页面语言切换。

---

## Tab 结构

| Tab | 内容 | 交互 |
|-----|------|------|
| Overview（总览）| 系统图解 + 三要素说明 + 双例对比 + Whole Syllable 例外 | 纯阅读 |
| Initials（声母）| 23 个声母按发音部位分 8 组 | 点击播放 |
| Finals（韵母）| 24 个韵母分 4 组（Simple/Compound/Front Nasal/Back Nasal）| 点击展开四声 + 播放 |
| Syllables（整体认读）| 16 个整体认读分 3 组 | 点击展开四声 + 播放 |
| Rules（规则）| 3 条书写规范 | 纯阅读 |

---

## Overview 设计要点

- 公式：`(Initial) + Final + Tone`（Initial 用虚线边框标注为可选）
- 双例对比说明零声母：
  - `b` + `ā` = **bā**（八）← 有声母
  - `∅` + `ān` = **ān**（安）← 零声母，Initial 位置为空
- 声调卡片：ToneContour 曲线图 + 文字描述
- Whole Syllable 黄色警告卡片：列出全部 16 个整体认读，说明不可拆分

---

## Finals 交互

点击韵母按钮（如 `ai`）后展开行内面板，显示四声带调拼音 + 逐个播放按钮：

```
[ai] ← 点击展开
┌─────────────────────────┐
│  āi [▶]  ái [▶]  ǎi [▶]  ài [▶] │
└─────────────────────────┘
```

同一时间只有一个展开（再点同一个则收起）。整体认读 Tab 同理。

## 音频系统

### 音频来源
从 `du.hanyupinyin.cn` 下载标准普通话教学录音，共 207 个 MP3 文件。

### 文件组织
```
frontend/public/audio/
├── initials/    # 23 个声母（无声调）：b.mp3, zh.mp3 ...
├── finals/      # 韵母（无调版 + 1-4 声版）：a.mp3, ai2.mp3, v1.mp3 ...
└── whole/       # 整体认读（1-4 声版）：zhi1.mp3, yi3.mp3 ...
```

### 命名规则
- `ü` 在文件名中统一用 `v` 表示（键盘输入惯例）：`v.mp3`、`ve2.mp3`、`vn3.mp3`
- 前端 `toAudioKey(pinyin)` 函数负责替换：`pinyin.replace("ü", "v")`

### 播放函数
```typescript
playInitial("b")          → /audio/initials/b.mp3
playFinalTone("ai", 2)    → /audio/finals/ai2.mp3
playFinalTone("ü", 3)     → /audio/finals/v3.mp3
playWholeTone("zhi", 1)   → /audio/whole/zhi1.mp3
```

无后端调用，直接浏览器播放静态文件，零延迟、零依赖。

### zh-Hans 翻译 Bug 修复（v3）
- **根因**：`translateTexts()` 写死 `from=zh-Hans`，当目标语言也是 zh-Hans 时，Azure 返回原文（英文）
- **修复**：`translateTexts(texts, toLang, fromLang = "zh-Hans")`，i18n 路由显式传 `"en"` 作为源语言

## 书写规则（3 条核心）

| # | 规则 | 说明 |
|---|------|------|
| 1 | Tone Placement（音调标注位置）| a/e 优先；其次 o；i 和 u 并列时标最后一个 |
| 2 | The ü Rule（ü 书写变化）| j/q/x/y 后去两点仍读 ü；键盘输入用 v |
| 3 | Apostrophe（隔音符号）| a/o/e 开头的音节接在其他音节后时用 ' 隔开（Xī'ān ≠ Xiān）|
