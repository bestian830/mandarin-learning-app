# v0.0.1 — "What's in your thought?"

## 功能

英文输入 → 三个结果同时展示：

| 字段 | 说明 |
|------|------|
| 简体中文 | Lingva Translate 翻译结果 |
| 繁体中文 | opencc-js 本地转换 |
| 拼音 | pinyin-pro 带声调拼音 |

**前端交互**
- 初始状态：Logo + 搜索框居中全屏（Google Search 风格）
- 搜索后：Logo 移至顶部，结果卡片出现在下方
- 示例搜索词 chip 快速触发查询
- Loading / Error 状态处理

## 技术栈

### Backend (`backend/`)
| 层 | 技术 |
|----|------|
| 运行时 | Node.js v23 + TypeScript |
| 框架 | Express v5 |
| 模块系统 | ESM (`"type": "module"`) |
| TS 执行 | tsx（`node --import tsx/esm`） |
| 翻译 | [Lingva Translate](https://github.com/thedaviddelta/lingva-translate) 公共 API — 免费，无需 key |
| 繁体转换 | [opencc-js](https://www.npmjs.com/package/opencc-js) — 本地简繁转换，无 API 调用 |
| 拼音 | [pinyin-pro](https://www.npmjs.com/package/pinyin-pro) — 汉字→带声调拼音 |
| 跨域 | cors |

### Frontend (`frontend/`)
| 层 | 技术 |
|----|------|
| 构建工具 | Vite v8 |
| 框架 | React 19 + TypeScript |
| 样式 | Tailwind CSS v4（`@tailwindcss/vite` 插件） |
| 字体 | Noto Serif SC（中文）/ Playfair Display（Logo）/ DM Sans（UI） |
| HTTP | 原生 `fetch`，通过 Vite proxy 转发到后端 |

## API

### `POST /api/search`

**请求**
```json
{ "query": "I love you" }
```

**响应**
```json
{
  "simplified": "我爱你",
  "traditional": "我愛你",
  "pinyin": "wǒ ài nǐ"
}
```

**错误响应**（HTTP 400 / 502）
```json
{ "error": "query is required" }
```

### `GET /health`
```json
{ "status": "ok" }
```

## 开发启动

```bash
# 后端（port 3001）
cd backend && npm run dev
# 或: ./backend/start.sh

# 前端（port 5173，/api/* 自动代理到 3001）
cd frontend && npm run dev
```
